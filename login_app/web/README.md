# Login Form App

Aplikasi Flutter dengan halaman login yang memiliki fitur:

## Fitur yang Diimplementasikan

✅ **Validasi Email** - Menggunakan regex untuk validasi format email  
✅ **Validasi Password** - Minimal 6 karakter  
✅ **Remember Me** - Checkbox untuk menyimpan kredensial  
✅ **Show/Hide Password** - Tombol untuk menampilkan/menyembunyikan password  
✅ **Loading State** - Indikator loading saat submit  
✅ **SharedPreferences** - Penyimpanan lokal untuk kredensial  

## Instalasi

1. **Clone atau download proyek**
2. **Tambahkan dependencies** di `pubspec.yaml`:
   ```yaml
   dependencies:
     flutter:
       sdk: flutter
     shared_preferences: ^2.2.2
